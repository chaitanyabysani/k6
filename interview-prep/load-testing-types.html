Load Testing Types

Complete reference — 8 test types with K6 patterns

Smoke Load Stress Spike Soak Breakpoint Volume Scalability
Test Type VUs Duration Load Pattern Primary Goal
Smoke
 1–2 1–5 min Minimal constant Verify basic functionality & script validity
Load
 10–100 5–30 min Gradual ramp up/down Measure normal & peak performance against SLAs
Stress
 100–500+ 10–60 min Increasing to breaking point Find capacity limits & failure behavior
Spike
 Sudden jump
e.g. 10 → 1000 → 10
 5–10 min Rapid increase/decrease Test surge handling (flash sales, viral events)
Soak
 50–100 Hours / Days Extended constant Find long-term issues: memory leaks, conn pool exhaustion
Breakpoint +
 ∞ (ramp until break) Until failure Continuous linear ramp Identify exact ceiling / max capacity of system
Volume +
 Moderate Variable Constant + large data payload Test behavior with massive data sets (millions of records)
Scalability +
 Incremental steps Variable Step-up with infra changes Validate performance as infra scales (pods, nodes, replicas)
